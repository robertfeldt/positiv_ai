# Positiv AI

> **AI för andra.**
> *En evidensgrundad katalog över AI som skapar för andra – inte fångar från oss.*

## Idé

AI-diskursen domineras av risk och förbud. Det som saknas är existensbevis med mekanism: konkreta, utvärderade exempel på AI som öppnar utåt — som bygger nyansering, förståelse, hälsa och, ytterst, fred — i stället för att kröka oss inåt i egna bubblor.

Varje exempel i katalogen måste svara på tre frågor:

1. **För vem?** Vilket värde skapas *för andra* (delaktighet, tillit, förståelse) — inte vad som fångas *från oss* (tid, data, uppmärksamhet)?
2. **Vilket designval?** Vilken mekanism (optimeringsmål, incitament, interaktionsdesign) gjorde att detta öppnar i stället för att fånga?
3. **Vilken evidens?** Hur är effekten utvärderad?

Det gör katalogen till en designmönstersamling — användbar för forskare, reglerare och upphandlare som behöver något positivt att kräva, inte bara något skadligt att förbjuda.

## Första exempel (under arbete)

- **Multi-persona viewpoint-analys för klimatförhandlingar** — AI som gestaltar olika aktörers perspektiv för att skapa nyansering och förståelse i Global Climate Summits (studie med Tove Engwall, kopplad till Paris-avtalets Global Stocktake).
- **LLM-stödd innehållsanalys av ensamhet** — AI som hjälper oss förstå ensamhetens mönster i svenska paneldata (SOM-institutet).
