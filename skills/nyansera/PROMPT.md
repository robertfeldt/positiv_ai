# Nyansera — fristående prompt

*Kopiera allt nedanför linjen, klistra in i valfri AI-chatt (ChatGPT, Claude, Gemini, Le Chat, …) och lägg din text sist.*

---

Jag vill att du hjälper mig att se mer, inte tycka mindre. Nedan ger jag dig ett argument, en ståndpunkt eller en frågeställning. Bredda och nyansera den i fem steg:

1. **Kärnan** — återge min position välvilligt i 2–4 meningar så jag ser att du förstått. Säg om du tolkat något osäkert.

2. **Dolda antaganden** — 2–4 antaganden som min position vilar på men inte uttalar. För varje: antagandet + vad som händer med positionen om det inte håller.

3. **Perspektiv** — 4–5 genuint olika perspektiv på frågan. De ska skilja sig längs verkliga axlar (värden, tidshorisont, levd erfarenhet, kunskapstradition, makt/position), inte vara kosmetiska varianter. Minst ett från någon som berörs men sällan hörs, och minst ett från en annan tidshorisont (kommande generationer eller historisk erfarenhet). För varje perspektiv, kort: vem · vad de värnar · hur frågan ser ut därifrån · vad de ser som jag inte ser · vad de skulle ge mig rätt i. Den som har perspektivet ska känna igen sig — inga halmgubbar.

4. **Starkaste motargumentet** — ETT motargument, formulerat så som dess bästa företrädare skulle formulera det.

5. **Nyanserad omformulering** — min position omformulerad med det som överlevt prövningen; säg vad som ändrats och varför. Avsluta med "Vad skulle få dig att ändra dig?": 2–3 konkreta observationer som borde få mig att ompröva.

Kvalitetskrav: ingen mesig relativism — skarpa oenigheter ska stå kvar skarpa. Tillverka inte balans där evidensen är sned. Smickra mig inte; jag ber om motstånd, inte bekräftelse. Hitta inte på fakta eller källor, och markera när ett perspektivs anspråk är empiriskt osäkra. Svara på samma språk som min text.
